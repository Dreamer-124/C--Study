# 知识回顾
## C++的一些主要优点是什么？
- 可以直接操作硬件，好的C++代码运行速度会很快，但是不好的C++代码运行速度会很慢，适应于高性能场景
- 直接从源代码通过编译器生成相应平台的可执行文件，具有良好的可移植性和泛化性

## C++是如何工作的？
- .cpp文件编写完 -> 将文件丢给编译器 -> 编译器对需要预处理的内容进行预处理 -> 编译器根据配置为每一个.cpp文件生成一个.obj文件 -> 链接器将所有的.obj文件合成为一个.exe文件 -> 执行.exe文件
